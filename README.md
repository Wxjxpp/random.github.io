# 随机数生成器 (Random Number Generator)

这是一个简洁、美观且功能强大的在线随机数生成器。

## 功能特点
- **多种生成模式**：支持单个、多个随机数生成。
- **美观的界面**：支持响应式设计，适配手机与电脑。
- **深色模式**：自动适配系统主题，也支持手动切换。
- **一言功能**：集成一言（Hitokoto） API，每次访问都有新惊喜。

## 如何使用
访问 [https://wxjxpp.github.io/random.github.io/](https://wxjxpp.github.io/random.github.io/) 即可直接使用。

## 本地开发
直接在浏览器中打开 `随机数生成器.html` 即可。
